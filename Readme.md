Multi-Agent AI Research System with LangChain

A lightweight AI research platform built with LangChain where multiple agents collaborate to perform research tasks. The system includes a Research Agent for information gathering, an Analysis Agent for insight extraction, a Verification Agent for fact-checking, and a Report Agent for generating final summaries.

Features
Multi-agent workflow
Automated research and analysis
Fact verification
Structured report generation
Easy to extend and customize
Tech Stack

Python, LangChain, OpenAI API, LangGraph (optional)

Run:
1. pip install -r requirements.txt
2. copy .env.example to .env and set OPENAI_API_KEY
3. python agents.py
